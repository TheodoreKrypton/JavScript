## Installation

### With Git
```bash
$ git clone https://github.com/TheodoreKrypton/javscript
$ npm install --only=prod
$ git submodule update --init --remote --recursive
$ cd ./frontend/ && npm install --only=prod && npm run build
```

## Usage
```bash
$ javpy
```
or
```bash
$ javpy --port 8081
```
