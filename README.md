## Installation

### With Git
```bash
$ git clone https://github.com/TheodoreKrypton/javscript
$ npm install -g --only=prod
```

### With npm
```bash
$ npm install -g javpy
```

## Usage
```bash
$ javpy
```
or
```bash
$ javpy --port 8081
```
